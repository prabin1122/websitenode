1. Executive Summary & Architecture

ShopHub is an enterprise-ready, modular e-commerce solution designed for scalability, speed, and sleek modern user experience. Built with Next.js (React 18), Express Node.js backend, and PostgreSQL powered by Prisma ORM, it offers full multi-channel shopping capabilities.

Layer Technologies Responsibilities
Frontend Storefront Next.js, TypeScript, Tailwind CSS, React Context SSR & CSR pre-rendering, Live Search Autocomplete, Slide-out Cart Drawer, Micro-animation toasts, Multi-step checkout.
Backend API Node.js, Express, Apollo GraphQL, Swagger REST API Endpoints (`/api/auth`, `/api/products`, `/api/orders`, `/api/payments`), Order fulfillment controller, Middleware auth guards.
Data & Persistence PostgreSQL, Prisma ORM, Redis Relational data modeling for Users, Products, Categories, Orders, OrderItems, Payments, and Coupons.
